# Mushroom Image Classification using CNN

This project focuses on classifying 4 mushroom species from image data using Convolutional Neural Networks (CNNs).

## Project Challenges

The classification task is challenging due to:

- Low image resolution (32×32 pixels)
- Small dataset size (1,200 images in total)
- High visual similarity between mushroom species in terms of shape and color

## Approach

To address these challenges, the project includes:

- Research on deep learning and transfer learning approaches, including ResNet and ImageNet-pretrained models
- Development of a lightweight CNN architecture for small-scale image classification
- Image augmentation techniques to improve generalization performance and reduce overfitting
- Model evaluation using classification metrics and confusion matrix analysis

## Technologies Used

- Scikit-learn
